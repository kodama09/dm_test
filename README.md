# User Registration API

Production-grade user registration and activation API for the Dailymotion technical assessment.

This repository is built incrementally, one roadmap subsection per commit. The current step initializes the Python project configuration with FastAPI, Pydantic, pytest, httpx, and Ruff.

## Requirements

- Python 3.12 or newer

## Development

Install the project dependencies with the optional development tools:

```bash
pip install -e ".[dev]"
```

Run Ruff once code is available:

```bash
ruff check .
```
